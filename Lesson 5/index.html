CSS-властивості.

CSS (англ. Cascading Style Sheets, укр. Каскадні таблиці стилів) — це спеціальна мова стилю сторінок, що використовується для опису їхнього зовнішнього вигляду. CSS використовується авторами та відвідувачами вебсторінок, щоб визначити кольори, шрифти, верстку та інші аспекти вигляду сторінки. За допомогою CSS можно покращити сприйняття та доступність контенту, забезпечити більшу гнучкість та контроль за відображенням контенту в різних умовах, зробити контент більш структурованим та простим, прибрати повтори тощо. CSS також дозволяє адаптувати контент до різних умов відображення (на екрані монітора, мобільного пристрою, у роздрукованому вигляді, на екрані телевізора, пристроях з підтримкою шрифту Брайля або голосових браузерах та ін.).

CSS-властивості шрифтів

Встановлює родину шрифту : font-family: "Ім'я шрифту", тип шрифту;

Типи шрифтів:

  • serif — шрифт з зарубками (засічками)
  • sans-serif — рублений (без зарубок)
  • cursive - курсивні шрифти
  • fantasy — декоративні
  • monospace — моноширинні
body {
	font-family: "CircularStd", serif;

 }

* успадковується

Встановлюєм розмір шрифту: font-size: цифрове значення одиниця виміру; 

.title {
	  font-size: 25px;

 }

* успадковується (за замовчунням 16px, якщо в браузері не вказано іншого)

Встановлюєм стиль шрифту: font-style: назва стилю;

  • normal (за замовчуванням) - звичайний (не змінений, стандартний) стиль
  • italic - курсив
  • oblique - нахилення літер
.title {
	  font-style: italic;

}

* успадковується

Встановлюєм насиченість (товщину) шрифту: font-weight: цифрове значення або значення словом;

  • 100 - thin
  • 300 - lite
  • 400 - Regular (normal) (за замовчуванням)
  • 500 - medium
  • 600 - semibold
  • 700 - bold
  • 900 - black
.title {
	  font-weight: 900;

}

* успадковується

CSS-властивості тексту

Встановлюєм колір тексту: color: слово або код кольору; 

.title {
	color: green;

	color: #1f7909;

	color: rgba(55, 168, 51, 0.5); - останне значення -прозорість. 

 }

* успадковується

Встановлюєм вирівнювання тексту: text-align: значення;

  • center - вирівнює по центру
  • left - вирівнює по лівому краю
  • right - вирівнює по правому краю
  • justify - вирівнює текст по ширині об'єкту в якому він знаходиться автоматично змінюючи пробіли між словами
.text {
	text-align: center;

 }

* успадковується (текстом)

Додає тексту певне оформлення : text-decoration: значення;

  • line-through - закреслює текст
  • overline - лінія над текстом
  • underline - лінія під текстом
  • none - (за замовченням) відміняє змінений раніше параметр
.text {
	text-decoration: underline;

 }

* не успадковується

Додає тексту тінь: text-shadow: горизонтальне_зміщення вертикальне_зміщення розмір колір; 

.text {
	text-shadow: 2px 2px 5px #000;

 }

* успадковується (текстом). Може бути мульти-тінь (значення через кому - text-shadow: 2px 2px 5px #000, 5px -2px 10px red;)

Встановлюєм тип (регістр) літер (прописні або заглавні): text-transform: значення;

  • capitalize - кожне слово починається з заглавної літери
  • lowercase - усі літери прописні (нижній регістр)
  • uppercase - усі літери заглавні (верхній регістр)
  • none - (за замовчунням) не змінює (або відмініє) значення
.text {
        text-transform: underline;

 }

* успадковується (текстом)

Визначаєм міжрядковий інтервал (відстань) (висота рядка): line-height: значення; 

.text {
	font-size: 10px;

	line-height: 1.2;

 }

* успадковується (текстом). Перевести пікселі в частини: значення line-height поділити на значення font-size

Встановлює відступ зліва у першого рядка тексту (Червона строка): text-indent: значення: 

.text {
	text-indent: 20px;

 }

* успадковується (текстом)

Встановлює інтервал між літерами (символами): letter-spacing: значення; 

.text {
	letter-spacing: 7px;

 }

* успадковується (текстом)

Управляє поведінкою пробілів між словами: white-space: значення;

  • nowrap - забороняє переніс рядка
  • normal - (за замовчунням) не змінює (або відмініє) значення
.text {
	white-space: nowrap;

 }

* успадковується (текстом)

Встановлює інтервал між словами: word-spacing: значення; 

.text {
        word-spacing: 20px;

 }

* успадковується (текстом)

CSS-властивості блоків

box-sizing. - Застосовується для зміни алгоритму обчислення величин.

  • content-box - (за замовченням) додає відступи до розміру об'єкту
  • border-box - значення розмірів враховують відступи
* {
    box-sizing: border-box;

}

* успадковується

Відступи

  • padding - Внутрішній відступ. Відступи ВСЕРЕДЕНІ об'єкту. Від кордонів об'єкту ДО ЙОГО КОНТЕНТУ.
    * НЕ успадковується
  • margin - Зовнішній відступ. Відступи ЗОВНІ об'єкту.
    * успадковується
    Варіанти запису значення:
  • padding: / margin: з_усіх_сторін;
  • padding: / margin: зверху_знизу зправа_зліва;
  • padding: / margin: зверху зправа_зліва знизу;
  • padding: / margin: зверху зправа знизу зліва;
  • -------------
  • padding-top: / margin-top: значення;
  • padding-right: / margin-right: значення;
  • padding-bottom: / margin-bottom: значення;
  • padding-left: / margin-lef: значення;

* padding не допускає негативного значення, якщо ви вказали його помилково, воно буде проігноровано.

* margin може бути заданий з негативним значенням.

Строчні і блочні об'єкти: Поведінка padding: / margin: суттєво відрізняеться!

    Відмінність Строчних від Блочних елементів у тому-що:
  • Рядковий елемент йдє в один рядок і займає розмір його вмісту.
  • Блочний елемент на веб-сторінці починається з нового рядка і йдє один під одним, має вигляд прямокутника і займає всю його ширину. Висота блочного елемента залежить від його вмісту.

ВАЖЛИВО ЗГАДАТИ:

  • width: auto; - (за замовченням) автоматичний розмір
  • width - чітко задає ширину блочного (блочно-строчного) об'єкту

  • * НЕ працює з строчними об'єктами
